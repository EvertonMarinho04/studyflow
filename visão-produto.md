# Visão do Produto

## Nome do sistema
StudyFlow

## Descrição
Sistema para organização de estudos que permite ao usuário cadastrar disciplinas, adicionar tarefas com prazos e dificuldades e identificar automaticamente quais atividades são mais urgentes.

## Usuários
- Estudante

## Funcionalidades
- Cadastro de disciplinas
- Cadastro de tarefas
- Definição de prazo
- Definição de dificuldade
- Visualização de tarefas
- Identificação de tarefas urgentes
