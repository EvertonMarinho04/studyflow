📌 3. Análise de Desempenho

Fatores que influenciaram:

Melhor entendimento do projeto na Sprint 2
Organização mais clara das tarefas no Kanban
Estimativas mais realistas
Redução de dúvidas técnicas

Possíveis dificuldades:

Falta de experiência inicial com GitHub
Tempo limitado para execução das tarefas
Ajustes no entendimento das histórias
📌 4. Ações de Melhoria
Melhorar estimativas de Story Points
Dividir tarefas grandes em tarefas menores
Manter organização do Kanban atualizada
Realizar revisões frequentes do progresso
Evitar acumular tarefas no final da Sprint
